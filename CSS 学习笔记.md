## 基本CSS选择器

* ### CSS 元素选择器

  元素选择器根据元素名称来选择 HTML 元素。

* ### CSS id 选择器

  id 选择器使用 HTML 元素的 id（**不可以数字开头**） 属性来选择特定元素。

  要选择具有特定 id 的元素，写一个＃，后跟该元素的 id。如**#abc{内容}    p id="abc"**

* ### CSS 类选择器

  类选择器选择有特定 class 属性的 HTML 元素。
  如需选择拥有特定 class 的元素，写一个句点（.）字符，后面跟类名。
